# Chat Amigo desde Consola con OpenAI

## Descripción
Este proyecto consiste en un chat sencillo en consola desarrollado en Python utilizando la API de OpenAI.  
El asistente fue configurado para responder como un “amigo”, con un tono cercano, amable y natural.

## Objetivo
Cumplir con la tarea de crear una aplicación básica que consuma un API de IA, funcione desde consola, esté documentada y se publique en GitHub sin exponer credenciales.

## Tecnologías utilizadas
- Python
- OpenAI API
- python-dotenv

## Requisitos
- Python 3.10 o superior
- Cuenta en OpenAI
- API key de OpenAI

## Configuración de OpenAI
Para utilizar la API, es necesario:

1. Crear una cuenta en OpenAI
2. Generar una API key desde el panel de usuario
3. Agregar créditos a la cuenta (en este caso se agregaron **$5 USD**) para poder realizar pruebas con la API bajo el modelo de pago por uso
